# MERCURY

A modern banking/fintech landing page built with **Vite**, **JavaScript**, and **Tailwind CSS**.
Includes smooth animations, responsive layout, and a small interactive feature using a public API.

## Features

* Responsive, modern UI inspired by fintech products
* Tailwind CSS styling with glass / liquid effects
* Scroll-based animations using ScrollReveal
* Mobile navigation with burger menu
* Yield calculator modal using a free FX API
* Scroll-to-top button
* Cookie consent stored in `localStorage`

## Tech Stack

* Vite
* JavaScript
* Tailwind CSS
* ScrollReveal.js
* Frankfurter API

## Getting Started

```bash
npm install
npm run dev
```
